# AfriConnect Summit 2026

## Auteur
- **Nom** : Mocpat
- **Prénom** : NEIGE
- **Classe** : L1 RT

## Description
Site vitrine pour la conférence tech panafricaine "AfriConnect Summit 2026", un événement annuel réunissant développeurs, entrepreneurs et investisseurs du continent africain.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, Variables CSS, animations)
- JavaScript Vanilla
- Git & GitHub Pages

## Fonctionnalités JavaScript implémentées
1. Dark Mode / Light Mode avec localStorage
2. Navbar dynamique (fond après 80px de scroll)
3. Menu hamburger mobile
4. Compte à rebours en temps réel
5. Compteurs animés au scroll
6. Animations fade-in avec IntersectionObserver
7. Onglets du programme (3 jours)
8. Filtrage dynamique des intervenants
9. Validation de formulaire complète
10. Bouton retour en haut
11. Année dynamique dans le footer
